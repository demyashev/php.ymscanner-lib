# About

Simple lib for [YM Сканнер](https://ymscanner.ru/) service.

# Install
Add these lines to the file `composer.json`

```
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/demyashev/php.ymscanner-lib"
        }
    ],
    "require": {
        "demyashev/php.review3-lib": "@dev"
    }
}
```
